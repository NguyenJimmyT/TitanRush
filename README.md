# TitanRush

###Instruction to run backend
Requirements:
python 3.10 or higher
``pip`` package manager

Steps:
* git clone main
* use the command ``pip install fastapi uvicorn requests python-dotenv pytz pydantic``
* To run it you type into the terminal ``uvicorn backend.server:app``

## How to build

### Requirements

* NodeJS version 20 or higher.
* Android Studio

### Steps

``git clone`` from the branch "Edward"

* Open a terminal/ bash shell/ powershell/ command prompt/ whatever your preference, and run:
``npm install`` to download the dependencies from node.
* Run ``npx expo install`` to install all react-native-expo files.
* Run ``npx expo start`` to start the dev server.

Once the dev server starts, press the "a" key in the terminal running the server, and it will begin automatically compiling a development build of TitanRush, and open it on your Android Studio simulator.

If there are issues with the android simulator, please follow the troubleshooting steps here:

https://docs.expo.dev/workflow/android-studio-emulator/
